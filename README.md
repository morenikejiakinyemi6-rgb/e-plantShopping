# e-plantShopping

Paradise Nursery is a React-based shopping application that allows users to browse 
and purchase a variety of houseplants.

## Features
- Browse houseplants organized by category
- Add plants to a shopping cart
- Update quantities and remove items from the cart
- View total cost before checkout

## Technologies Used
- React
- Redux Toolkit
- CSS

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
